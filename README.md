# MakeSense

MakeSense is the hardware and software toolkit for the EuroHaptics 2026
workshop on data-driven haptic texture reproduction.

## Repository layout

- [`host/`](host/) — PyQt6 application for recording, modelling,
  characterization, compensation, and rendering.
- [`device/firmware/stm32_haptic_device/`](device/firmware/stm32_haptic_device/)
  — active STM32 firmware.
- [`device/pcb_files/`](device/pcb_files/) — KiCad sources and manufacturing
  outputs for the MakeSense hardware.
- [`Documents/`](Documents/) — workshop flyer, kit guide, schedule, and banner.

See the [host README](host/README.md) for installation and application usage.
