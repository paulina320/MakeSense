# Quick Start Guide

## Installation (5 minutes)

### Linux/Mac
```bash
cd "haptic-workshop"
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python main.py
```

### Windows
```bash
cd "haptic-workshop"
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python main.py
```
